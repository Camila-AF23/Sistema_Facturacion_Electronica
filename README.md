# Sistema Facturación Electrónica
## Proyecto de sistema de facturación electrónica y gestión de inventario 💰📄💾
Hoy en día es muy común pedir la factura electrónica en muchas tiendas y marcas gigantes cuando compramos en ellas, pero qué pasará dentro de unos años cuando esta sea un requisito para evitar sanciones o multas en los comercios pequeños como los que tenemos cerca de nuestra casa, pequeños negocios familiares como un mini-mercado, panadería o fruver.

Para resolver este problema a futuro está este sistema de facturación electrónica para tiendas locales (en su fase de prueba) 😎.
